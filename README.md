# FastApiSkelton

Minimalny generator szkieletu aplikacji FastAPI dla Alpine Linux z usługą OpenRC.

## Instalacja

Uruchom na Alpine jako root:

```sh
wget -qO- https://raw.githubusercontent.com/SP5YAM/FastApiSkelton/main/setup-fastapi-alpine.sh | sh
```

Skrypt poprosi o nazwę aplikacji i port HTTP, a następnie utworzy usługę oraz środowisko aplikacji.
